 # AndroidMu 做VA兼容工具 
 工具主要是查找不同Android 版本 中某个类是否有某属性、方法等，工具另附带各版本android 下载
     使用方式：
         1、查找某个类是否有属性   ./find_compare.sh ActivityManager.java START_TASK_TO_FRONT
         2、下载android 源码      ./find_compare.sh clone  （脚本中配置下载项需自行修改）
